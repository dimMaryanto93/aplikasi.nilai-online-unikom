Programan Aplikasi Nilai Online Universitas Komputer Indonesia anda dapat mengakses semua modul yang terdapat pada menu utama.Beberapa modul yang terdapat pada Sistem Informasi Nilai Online berbasiskan WEB.

Aplikasi ini dirancang agar dapat membantu mahasiswa, dosen. untuk sekedar melihat history nilai yang telah diraih mahasiswa atau melakukan pengolahan data nilai yang hanya dapat dilakukan oleh dosen dan administrator.

  1. Data Dosen ,yang berfungsi sebagai untuk mengelola data Dosen (read ,input, update, delete)
  2. Data Mahasiswa ,yang berfungsi sebagai untuk mengelola data Mahasiswa (read ,input, update, delete)
  3. Data Matakuliah ,yang berfungsi sebagai untuk mengelola data Matakuliah (read ,input, update, delete)
  4. Data Nilai ,yang berfungsi sebagai untuk mengelola data Nilai (read ,input)
